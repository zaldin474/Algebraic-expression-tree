# Algebraic Expression Tree

A Java program that parses and evaluates algebraic expressions using a **binary expression tree**.

## Features
- Builds expression trees from prefix or postfix notation
- Evaluates expressions with variable substitution
- Reads input from files
- Handles multiple test cases from sample input files

## Concepts Demonstrated
- Binary tree construction and traversal
- Recursive algorithms
- File I/O in Java
- HashMap for variable mapping

## How to Run
```
javac src/*.java -d bin/
java -cp bin Main sample_input1.txt
```
